# Portfolio Images

Please add your portfolio images to this directory with the following names:

- `bedroom.jpg` - Bedroom renovation project
- `bathroom1.jpg` - First bathroom project
- `bathroom2.jpg` - Second bathroom project

You can also add more images and update the portfolio section in `app/page.tsx` to include them.

## Image Requirements

- Recommended size: 1200x800 pixels or larger
- Format: JPG, PNG, or WebP
- File size: Keep under 2MB for optimal loading
